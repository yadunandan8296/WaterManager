# 💧 Water Manager APK

## QUICKEST WAY TO GET YOUR APK (GitHub Actions - Free, No Install Needed)

1. Go to https://github.com/new  → create a FREE account if you don't have one
2. Create a new repository called "WaterManager" (set it to Public)
3. Upload this entire zip folder to the repository
4. GitHub will automatically build the APK
5. Go to Actions tab → click the latest run → download "WaterManager-debug" artifact
6. Unzip it → install app-debug.apk on your phone

That's it! No Android Studio needed.

---

## Alternative: Android Studio

1. Install Android Studio from https://developer.android.com/studio
2. Open this project folder
3. Wait for Gradle sync to finish
4. Go to Build → Build APK
5. APK will be at: app/build/outputs/apk/debug/app-debug.apk
